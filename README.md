# Azure Hybrid Identity & Governance Lab

A hands-on lab simulating a hybrid identity environment with on-prem AD DS 
synced to Microsoft Entra ID, combined with RBAC, Azure Policy, and 
Conditional Access — built as part of my AZ-104 exam preparation.

## 🎯 Goals
- Practice Management Group / RBAC / Governance concepts hands-on
- Document a realistic hybrid identity setup
- Build a portfolio project for Cloud Security roles

## 📂 Structure
- `docs/` – step-by-step documentation with screenshots
- `scripts/` – PowerShell / CLI scripts used in the lab
- `diagrams/` – architecture diagrams
